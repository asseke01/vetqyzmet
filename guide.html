Руководство пользователя

1. Основные сведения о Едином портале

Единый портал предназначен для автоматизации процесса предоставления услуг населению (физическим и юридическим лицам) соответствующими организациями.
Мобильное приложение позволяет каждому владельцу сельскохозяйственного животного:

Реализация данного портала основана на решениях, использующих принципы «одного окна», что обеспечивает прозрачность процессов обслуживания, сокращает сроки оказания услуг и упрощает процесс работы для ветеринарных специалистов.

2. Что нужно для работы
Для работы с порталом необходимо:

  • Действующая ЭЦП (электронная цифровая подпись);
  • Установленная и запущенная программа NCALayer

Как получить ЭЦП можно узнать, пройдя по ссылке https://pki.gov.kz/или обратившись в ближайший ЦОН (центр обслуживания населения).
Как установить и запустить для работы программу NCALayer можно узнать, пройдя по ссылке http://pki.gov.kz/index.php/ru/ncalayer.
